# Changelog

## v5.1.0 — 隊伍名稱自動組合

- 事件／比賽名稱改為左右兩個隊伍輸入框，中央 `vs` 固定顯示。
- `vs` 前方名稱自動作為體方，後方名稱自動作為用方，事件名稱同步組合，不再重複輸入。
- 新增體用名稱必填與自動組合測試。

## v5.0.1 — Streamlit Community Cloud 啟動修正

- 固定 Streamlit 1.56.0，避開新版 Starlette／Uvicorn 啟動後的原生程序崩潰。
- 新增真實 Streamlit CLI 與 `/_stcore/health` 啟動測試，不再只依賴 AppTest。

## v5.0.0

- 產品由足球比分預測系統改為「完整排卦系統」。
- 移除 AI、足球先驗、比分網格、規則校準、命中率、相似案例與賽後流程。
- 輸入保留體方段落、用方段落與完整中性段落；補充資料只保存、不取數。
- 輸出新增可稽核計數、餘數、本卦六爻表、互卦取爻、動爻陰陽反轉與變卦完整結構。
- 新增 JSON／Markdown 排盤下載與獨立 `meihua_castings.csv` 儲存。
- 八卦資料庫改為傳統結構欄位，移除足球攻守與進球欄位。
- 六十四卦資料庫擴充為 64 卦、384 爻全覆蓋，包含卦辭、彖傳、大象、爻辭與小象。
- 乾坤加入用九／用六與《文言》，另加入《說卦》《繫辭》《序卦》《雜卦》。
- 每卦新增互卦、錯卦與綜卦索引。
- 舊比分案例與報告保留為唯讀歷史資料，不進入 v5 流程。
