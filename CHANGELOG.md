# Changelog

## v4.0.0

- 改為「純足球先驗先建立 λ，卦象只作單方 ±25% 有界修正」。
- 新增 `football_prior.py` 與獨立足球比分、勝平負機率。
- 比分網格由單方 0–5 擴至 0–10，移除高比分硬性折價。
- 單場賽果規則全部降為 `hypothesis`，預測權重為 0。
- 新增 reviewed／verified 升級門檻。
- 新增 SHA-256 預測雜湊、版本序號、前版連結與內容雜湊報告。
- 相同預測冪等儲存；不同預測新增版本；賽後欄位不可被重新預測清空。
- 新增足球基線、卦象規則、AI 與最終排序的分層評估。
- 新增 Brier、LogLoss、BTTS、大小 2.5 與 v4 鎖定盲測。
- 修正沒有 AI 輸出時被錯算為 AI 未命中。
- 修正舊比分欄位在 dashboard 未被正確讀取。
- 同一組對賽即使隊名順序相反，也不會被當成本場歷史案例。
- 移除 Streamlit 入口的 runtime monkey patch，決策控制收斂至單一實作。
- README、架構、資料、模型、規則、評估、操作與 CI 文件完整化。
- 測試擴充至足球先驗、資料鎖定、規則治理、指標與 Streamlit 冒煙測試。
