# 資料來源

## 排卦輸入

排卦引擎只讀取使用者貼入的三段文字；表單不再提供賽前補充資料輸入框。

## 經文資料

古典經文與結構化資料的來源、授權及完整性界線見 `knowledge/SOURCES.md`。
《焦氏易林》採完整標點顯示本，只作原典查閱與排盤附件，不進入文字計數，也不由程式產生事件解讀。排卦 JSON 只附上本次「本卦之變卦」對應的一條林辭及其來源資訊，不把林辭轉換成判斷。

## 禁止輸入排卦計算的資料

- 足球實力評分、排名、傷停權重。
- 勝負、比分、賠率、期望進球或 Poisson 數值。
- 歷史案例的預測或實際賽果。
- AI 產生的解釋。
- 賽後數據。

這些欄位在 v5 主程式中不存在。
