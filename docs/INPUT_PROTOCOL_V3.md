# 球隊自述起象輸入 v3

## 用途與證據邊界

這套格式只處理文字輸入的一致性、可重複性、可稽核性與回測品質。搜尋截至 2026-07-19，沒有找到針對「球隊第一人稱自述字數起卦」的公開實證研究，因此不可宣稱某一種寫法本身能提高梅花易數或足球預測準確率。

足球預測研究主要使用歷史資料、評分模型、賠率與群體資訊；預註冊研究方法則主張在知道結果前固定設計與分析方案，以降低事後選擇和研究者自由度。這些研究只支持「固定規格、保存原文、留出回測」的研究方法，不驗證梅花易數的預測效力。

- [The Betting Odds Rating System: Using soccer forecasts to forecast soccer](https://doi.org/10.1371/journal.pone.0198668)
- [A manifesto for reproducible science](https://doi.org/10.1038/s41562-016-0021)
- [The past, present and future of Registered Reports](https://doi.org/10.1038/s41562-021-01193-7)

## 十一行固定格式

```text
我是XXX。
目前我的客觀狀態……
我的士氣與比賽壓力……
我的預計比賽策略……
我主要依靠的組織支點……
我的主要進攻通道……
我的主要防守結構……
我最大的相對優勢……
我目前最明顯的限制……
我最需要防範對手的……
我希望在九十分鐘內……
```

每項必須各自成為一個非空行，順序與固定開頭不可更改。空白行不計入十一行。

## 欄位寫作標準

1. 客觀狀態：陣容完整度、主要傷停、休息旅程、輪換或近期賽前攻守訊號；避免只寫「穩定」。
2. 士氣與壓力：只用有賽前依據的描述；無明確資訊可標成中性。
3. 預計策略：只寫主要計畫，不把所有可能戰術塞入同一句。
4. 組織支點：優先寫角色和機制，球員姓名只能作輔助。
5. 進攻通道：寫最可能形成射門的路徑，而非泛稱配合或突破。
6. 防守結構：同時交代站位高度、區域與保護位置。
7. 相對優勢：必須相對於本場對手，不寫球隊永久標籤。
8. 自身限制：獨立記錄傷停、體能、終結、出球、回防或深度問題。
9. 對手威脅：只寫最重要的一至兩個對手威脅，不與自身限制混用。
10. 九十分鐘目標：寫可執行行為，不寫取勝、晉級或復仇願望。

## 防噪硬規則

- 體、用雙方資訊結構完全對稱。
- 每行只承載一個主要訊號。
- 自身限制與對手威脅必須分開。
- 禁止必勝、希望取勝、爭取晉級、復仇、創造歷史等結果導向詞。
- 只使用賽前已知資訊，固定判斷九十分鐘，不含延長賽與 PK。
- 建議開賽前六小時凍結；重大傷停或先發變化才重做，且雙方一起更新。
- 起卦後保存原文與版本，不因卦象不合直覺替換同義詞、增刪句子或重新計數。

控制詞彙目前只作一致性提示，不作強制封閉字典：狀態可優先用「明顯正面、略正面、中性、略負面、明顯負面」；主要策略可優先用「主動控球、快速轉換、直接推進、中低位防守」。若未來要擴充封閉詞彙，必須先完成完整詞表與回測規格，避免半套字典反而產生新偏差。

## 與足球先驗的邊界

本系統 v5 只負責排卦與卦義資料，不輸出勝平負、比分或機率，因此本版不新增賠率、排名、Poisson 或足球先驗欄位。若未來另建預測層，應與起卦層完全分離：足球先驗負責基礎實力與進球範圍，卦象只作賽事劇本和候選排序的獨立研究訊號，兩層都必須保存賽前版本並使用留出賽事評估。
