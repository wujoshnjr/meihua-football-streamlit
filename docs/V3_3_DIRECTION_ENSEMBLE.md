# v3.3 方向證據融合引擎

## 要解決的問題

v3.2 可能出現三個比分全部押同一方向，原因是：

1. `體生用`、`用剋體`等單一關係對期望進球修正過大。
2. 動爻轉象被當成整場完全替換，例如體方由離轉艮時，把前段火力一起刪除。
3. AI只能從規則前12名挑比分；若前12名全偏同一方，AI沒有糾錯空間。
4. 案例少於3場時一律禁止AI改變方向，即使賽前足球證據非常強。
5. 足球實力、卦象證據與歷史案例沒有分層記錄，難以知道錯在哪一層。

## v3.3 原則

- 體用生剋是風險訊號，不是勝負公式。
- 本卦、互卦、動爻、變卦按前中後段解讀；後段收束不代表前段不能進球。
- 新增體方/用方賽前實力、可信度與場地，全部不參與起卦字數。
- 規則候選池強制保留體勝、平局、用勝各至少4個代表比分。
- AI先獨立評估足球證據，再評估卦象證據，最後明確處理矛盾。
- 弱證據不得翻盤；只有證據品質、方向信心、實力差同時達標，才可在平衡候選池前9名內糾正規則方向。
- 最終報告保存勝平負機率、足球先驗、AI證據品質、矛盾警告與控制模式。

## 不做的事

- 不把法國2-0摩洛哥或任何單場結果寫成固定規則。
- 不使用本場賽後資料產生賽前預測。
- 不保證命中或獲利。
- 不連接任何真實投注或資金系統。

## 驗證

`tests/test_v33_direction.py`覆蓋：

- 體生用不再機械導致三個用方勝。
- 候選池包含體勝、平局、用勝。
- 弱AI證據不能翻盤。
- 強且可驗證的足球證據可以在受控候選池內糾正方向。
