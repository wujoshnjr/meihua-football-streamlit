# 條件式卦義規則

## 目的

同一經卦本來就有多個類象。條件式卦義庫不把「艮＝低比分」「震＝多球」之類單句當成固定答案，而是先由整條卦線建立客觀訊號，再排序該卦本次較符合的義項。

## 三層資料

1. 古典核心：依《說卦》與《梅花易數》八卦類象保存，例如乾健、坤順、震動、巽入、坎陷、離麗、艮止、兌說。
2. 足球可能義：每個經卦固定八個正反義項，說明在足球場景可能表現為進攻、防守、節奏、轉換或失效。
3. 條件規則：每個經卦固定六條規則，用可稽核訊號決定哪些義項提高或降低權重。

## 固定訊號

- 變前／變後旺衰：旺、相、休、囚、死。
- 變前／變後生克：體生用、用生體、體克用、用克體、比和。
- 動爻：落在體或用、初至四爻或五上爻、旬空、月破。
- 結構：本卦納甲六沖與六合數量。
- 卦線：本卦、互卦、變卦及體用轉象是否具有突破、滲透、危險轉換、封鎖或鏡像訊號。

## 排序方法

一條規則只有在 `all` 全部成立、`any` 至少一項成立且 `none` 全不成立時才命中。命中後提高 `prefer` 義項、降低 `suppress` 義項；規則優先級只用於同一卦內排序。前三義不是預測機率，也不直接換算進球或比分。

如果沒有規則命中，系統保留全部可能義，不強行選義。每次排卦與 HTML 報告都會顯示命中條件、證據與全部候選，方便人工覆核。

## 艮的指定規則

- 先得勢且變後旺相或受生：守成、控制節奏。
- 高動能卦變艮且生克轉差：攻勢中斷、停滯。
- 艮不變且有生扶／優勢：防守穩定、單點屏障。
- 艮受克並轉弱、旬空或月破：被困住、停滯。
- 整條卦線缺少破門通道：封鎖、節奏收束；只提高低機會環境，不直接指定 0:0。
- 對方具有震、巽、離、夬、噬嗑等突破訊號：艮也可能是被突破的屏障。

## 方法界線

足球條件規則是本專案可回測的應用層，不是《周易》或《梅花易數》原文。規則必須在賽前固定；單一賽果只能提出待回測假說，不能事後改寫同一場的條件。
