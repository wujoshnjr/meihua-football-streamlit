# Model Card：梅花易數足球 AI 系統 v4

## 模型類型

混合研究系統：人工足球先驗、獨立 Poisson 比分基線、梅花易數有界修正、歷史案例檢索與選用的 GitHub Models 重排。

## 預期用途

- 保存可重現的 90 分鐘賽前預測。
- 比較足球基線與卦象修正是否有前瞻增益。
- 累積賽後校準案例與研究假說。
- 研究 AI 在受限候選池中的排序價值。

## 非預期用途

- 真實投注、自動下注或資金配置。
- 宣稱保證獲利、保證比分或科學證實梅花易數有效。
- 使用賽後資料生成看似賽前的預測。
- 醫療、法律、財務或其他高風險決策。

## 輸入

- 三段起卦文字。
- 不計字數的賽前補充資料。
- 體／用方實力 0–100、可信度 0–1、場地。
- 已確認的歷史案例。

## 輸出

- 本卦、互卦、動爻、變卦與體用關係。
- 足球基線與修正後雙方 λ。
- 勝平負機率、比分網格與三個候選。
- AI 證據、控制模式與最終排序。
- 預測雜湊、版本與賽後評估。

## 主要假設

- 人工實力分能粗略表達賽前相對強弱。
- 進球可用獨立 Poisson 作為簡化基線。
- 卦象若有資訊價值，應在有界範圍內改善基線，而不應無限制主導。
- 歷史案例只有在賽前版本與賽後校準完整時才有研究價值。

## 限制與風險

- 實力分可能受輸入者偏誤影響。
- 獨立 Poisson 忽略比分狀態、紅牌與雙方進球相關性。
- 小樣本精確比分命中率波動極大。
- 歷史案例與規則可能產生選擇偏誤、資料洩漏與事後擬合。
- AI 可能生成流暢但缺乏證據的理由，因此受到 JSON schema、候選池與方向保護限制。
- GitHub Models 的可用模型與額度可能變動。

## 風險控制

- 只判 90 分鐘。
- 同場案例排除。
- 單場規則權重 0。
- 卦象倍率 ±25%。
- 預測版本不可覆寫。
- 足球、卦象、AI 分層評估。
- AI 失敗自動回退。
